# README

# Requirement

* ruby version: 2.6.7
* rails version: 5.2.8
* database: Postgres

# running test cases

1. bundle install
2. rails db:create db:migrate
3. rspec or rspec specs/

## running server

1. bundle install
2. rails db:create db:migrate db:seed
3. rake assets:precompile

# Time taken to build this project

* around 6 hours
